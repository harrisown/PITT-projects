Introduction
This project finds the largest common sequence between two different strings. The algorithm's implemented use dynamic programming and memoization in order to quickly compute results and store subproblems. 

Installation

The following files should be present
TestLCS.java
LCS.java

Compile with:
javac TestLCS.java

Useage
This program should compile and run with JAVA 2 SDK1.5 or later

run with java TestLCS
